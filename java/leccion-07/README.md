# HolaMundo en Java (7)

## Conceptos

 - collecciones
   - sets
   - listas
   - mapas
 - Java ARchives (JAR)  

## Comandos

```
$ javac -d classes com/acme/*.java com/acme/greetings/*.java
$ cd classes
$ java com.acme.Main

$ jar -c -f ../greet.jar -e com.acme.Main .
$ cd ..
$ java -jar greet.jar
```
